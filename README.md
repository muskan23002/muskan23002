## 📂 Featured Projects

### 🛍️ [Multi-Brand Beauty Marketing Campaign Analysis](https://github.com/muskan23002/multi-beauty-brand-marketing-analysis)
> SQL • PostgreSQL • Marketing Analytics

End-to-end analysis of **1,66,665 marketing campaigns** across Nykaa, Purplle & Tira.
Answered 10 real business questions using advanced PostgreSQL — CTEs, Window Functions, UNNEST, RANK, STDDEV, and a CTR-CVR-ROAS Performance Matrix.

**Key Finding:** Tira's Youth + Instagram combination delivered the highest ROAS in the entire dataset. Google ranked last for all 3 brands — proving beauty is a visual discovery category.

`PostgreSQL` `CTR` `CVR` `ROAS` `Window Functions` `CTE` `Marketing Analytics`

### 📊 [Amazon Retail Sales Analysis](https://github.com/muskan23002/eda-amazon-retail-sales-excel)
> Excel • Power Query • EDA

Exploratory Data Analysis of **1,000 Amazon transactions** across 3 months using Microsoft Excel.
Built KPI dashboards, pivot analysis, and time-based sales breakdowns using Power Query and Pivot Charts.

**Key Finding:** Afternoon hours drove 52% of all transactions. Female customers contributed 52% of total revenue. Health & Beauty was the top revenue-generating product line.

`Excel` `Power Query` `Pivot Tables` `KPI Dashboard` `EDA`
